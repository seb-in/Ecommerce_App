# Ecommerce App - Developing Stage

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.7.

# For API
### Getting started
 Install node_modules
 
  ```npm install``

Install JSON Server

  ```npm install -g json-server```

Start JSON Server

  ```json-server --watch db.json```
